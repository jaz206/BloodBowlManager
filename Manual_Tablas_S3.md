## 📈 1. PROGRESIÓN Y VALORACIÓN

### Tabla de Mejora de Jugadores
| Mejora | Azar | Elegida |
| :--- | :---: | :---: |
| **Habilidad Primaria** | 3 PE | 6 PE |
| **Habilidad Secundaria** | - | 12 PE |
| **Mejora de Atributo** | 14 PE (1D8) | - |

### Aumento de Valoración (VAE / CTV)
| Atributo/Mejora | Coste |
| :--- | :--- |
| **Habilidad Primaria** | +20,000 |
| **Habilidad Secundaria** | +40,000 |
| **+1 AV (Armadura)** | +10,000 |
| **+1 MV (Movimiento)** | +20,000 |
| **+1 PA (Pase)** | +20,000 |
| **+1 AG (Agilidad)** | +30,000 |
| **+1 FU (Fuerza)** | +60,000 |

> [!IMPORTANT]
> **Habilidades de Élite (Temporada 3):** Las habilidades **Placar (Block)**, **Esquivar (Dodge)**, **Defensa (Guard)** y **Golpe mortífero (Mighty Blow)** tienen un recargo adicional de **+10,000 MO** sobre su coste normal de mejora.

## 🎲 2. TABLAS DE JUEGO

### Clima (2D6)
| 2D6 | Resultado | Efecto |
| :---: | :--- | :--- |
| **2** | **Calor Asfixiante** | 1D3 jugadores se desmayan tras la entrada (van a Reservas). |
| **3** | **Muy Soleado** | -1 a los chequeos de Pase. |
| **4-10** | **Clima Perfecto** | Sin efectos. |
| **11** | **Lluvioso** | -1 a Recoger, Atrapar e Interceptar. |
| **12** | **Ventisca** | -1 a Forzar la Marcha (GFI). Solo pases rápidos o cortos. |

### Eventos de Patada Inicial (2D6)
| 2D6 | Evento | Efecto Resumido |
| :---: | :--- | :--- |
| **2** | **Árbitro Intimidado** | Cada equipo recibe 1 Soborno gratuito. |
| **3** | **Tiempo Muerto** | Se suma/resta 1 turno al marcador de ambos. |
| **4** | **Defensa Sólida** | El pateador redespliega 1D3+3 jugadores. |
| **5** | **Patada Alta** | El receptor coloca un jugador bajo el balón. |
| **6** | **Hinchas Animan** | Ganador de tirada (1D6+Animadoras) gana apoyo ofensivo. |
| **7** | **Entrenador Brillante** | Ganador de tirada (1D6+Ayudantes) gana 1 RR extra. |
| **8** | **Clima Cambiante** | Nueva tirada de clima. Si es perfecto, el balón se escora (3). |
| **9** | **Anticipación** | El receptor mueve 1D3+3 jugadores (1 casilla). |
| **10** | **¡A la Carga!** | El pateador mueve 1D3+3 jugadores (1 puede hacer Blitz). |
| **11** | **Indigestión** | Tirada enfrentada: el perdedor tiene un jugador con -1 MV/AR. |
| **12** | **Invasión de Campo** | Tirada enfrentada: 1D3 jugadores del perdedor Aturdidos. |
